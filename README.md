## 🐤 BByark - 삐약
학급회 회의록 작성 및 제출을 간단하게 해주는 서비스

### 프로젝트 시작
> 클론을 받은 후 .env 파일을 만들어 mongodb atlas url을 넣어야합니다.


    git clone https://github.com/BByark/Server-Nestjs.git
    npm install
    npm run start:dev
